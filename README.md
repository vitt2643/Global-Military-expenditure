# Global Military-Industrial Base Analysis

## Overview

The Global Military-Industrial Base Analysis is an end-to-end Power BI analytics project that explores how military expenditure, defense manufacturing, and industrial capacity have evolved across countries over time.

Rather than focusing only on defense spending, this project combines multiple datasets to uncover relationships between national military investment, defense manufacturers, regional trends, and long-term industrial growth. The dashboard enables users to move beyond static reporting through interactive visualizations and diagnostic analytics.

## Objectives

- Analyze global military expenditure across countries and regions
- Identify long-term defense spending trends
- Examine relationships between military budgets and domestic defense industries
- Compare regional defense investment patterns
- Detect spending volatility and emerging industrial powerhouses
- Deliver an interactive dashboard for exploratory analysis

## Dataset Sources

- SIPRI Military Expenditure Database
- Global Military Spending Dataset
- Top Defense Manufacturers Dataset

These datasets were integrated to create a unified analytical model linking sovereign military expenditure with defense industry performance.

## Data Preparation

The project required extensive preprocessing before visualization:

- Standardized country names across datasets
- Built a country mapping table
- Cleaned inconsistent text values
- Removed invalid and missing financial records
- Converted spending values into fixed decimal format
- Applied Power Query transformations
- Created a Star Schema data model for scalable analytics

## Data Model

The dashboard is built using a Star Schema consisting of:

- Country Dimension
- Military Spending Fact Tables
- Defense Manufacturer Fact Table
- Regional Attributes
- Time Dimension

This structure supports efficient filtering, cross-analysis, and interactive reporting.
